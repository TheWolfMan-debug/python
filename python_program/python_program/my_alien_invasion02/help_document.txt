菜单快捷键：
s	--------	Play
n	--------	Play new game
q	--------	Quit the game
h	--------	Help
p	--------	pause
操作快捷键：
space	--------	Firing bullets
up arrow	--------	Move up
down arrow	--------	Move down
left arrow	--------	Move left
right arrow	--------	Move right
